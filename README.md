## 介绍
最初由风扇化、星辰ran构思于`2023`年：
[关于米雪工艺的记载-网易大神]()

<br>
<br>早期的米雪工艺使用`Minecraft AddOns Maker(MAM)`制作而成，
<br>最后版本达到了`1.5 beta`，之后米雪工艺的制作停滞了很长时间。
<br>
<br>现在重启米雪工艺计划，并弃用`MAM`，改用
__手敲键盘读文档__
的方式<br>
